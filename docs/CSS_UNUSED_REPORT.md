## Рекомендации

### Можно безопасно удалить

Все классы и ID из списка "Неиспользуемые" можно безопасно удалить из CSS, так как они не используются ни в статическом HTML, ни в JavaScript коде.

### Требуют дополнительной проверки

1. **`.badge-level`, `.badge-progress`, `.badge-progress-fill`, `.badge-progress-row`, `.badge-title`** - Эти классы могут быть частью старой реализации badge-кнопок. Проверьте, не используются ли они в других версиях приложения или в планах на будущее.

2. **`.btn--with-bg-image`** - Этот класс определен, но используется только через ID селекторы. Возможно, стоит удалить класс и оставить только ID селекторы.

3. **`.brand`, `.brand-title`, `.logo`** - Эти классы могут быть частью старой версии главной страницы. Проверьте историю изменений.

4. **`.story-hero-content`, `.story-hero-title`** - Эти классы определены, но не используются. Возможно, планировались для будущей функциональности.

5. **`.trophy-*` классы** - Многие классы для трофеев определены, но не используются. Проверьте, не планируется ли их использование в будущем.

## Примечания

- Некоторые селекторы могут использоваться через составные селекторы (например, `.badge-btn .badge-level`), но если сам класс `.badge-level` нигде не присваивается, то такой селектор никогда не сработает.

- Некоторые классы могут быть зарезервированы для будущего использования. Перед удалением рекомендуется проверить планы разработки.

- При удалении неиспользуемых правил рекомендуется делать это постепенно и тестировать приложение после каждого удаления.
